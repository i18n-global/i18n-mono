# v1.4.0 Release Notes

**Release Date:** January 15, 2025

## üéØ Initial Major Features

This release marks the first major version of i18nexus-tools with core functionality for automated i18n workflows.

### ‚ú® Core Features

#### Project Initialization

- **NEW**: `i18n-sheets init` command for project setup
- **NEW**: Automatic creation of `i18nexus.config.json`
- **NEW**: Automatic creation of `locales/en.json` and `locales/ko.json`
- **NEW**: Google Sheets integration setup (optional)

#### String Wrapping

- **NEW**: `i18n-wrapper` command for automatic string wrapping
- **NEW**: Korean and English string detection
- **NEW**: Automatic `useTranslation` hook injection
- **NEW**: Support for JSX elements and expressions

#### Translation Extraction

- **NEW**: `i18n-extractor` command for key extraction
- **NEW**: Multi-language file generation (en.json, ko.json)
- **NEW**: CSV export for Google Sheets compatibility
- **NEW**: Duplicate key detection and reporting

#### Google Sheets Integration

- **NEW**: `i18n-sheets` command suite
- **NEW**: Upload translations to Google Sheets
- **NEW**: Download translations from Google Sheets
- **NEW**: Bidirectional sync capabilities
- **NEW**: Status checking and validation

### üîß Configuration

#### Configuration File

```json
{
  "languages": ["en", "ko"],
  "defaultLanguage": "ko",
  "localesDir": "./locales",
  "sourcePattern": "src/**/*.{js,jsx,ts,tsx}",
  "googleSheets": {
    "spreadsheetId": "",
    "credentialsPath": "./credentials.json",
    "sheetName": "Translations"
  }
}
```

### üìù Basic Workflow

#### Initial Setup

```bash
# Initialize project
npx i18n-sheets init

# Optional: Add Google Sheets integration
npx i18n-sheets init -s <spreadsheet-id>
```

#### Development Workflow

```bash
# Wrap hardcoded strings
npx i18n-wrapper

# Extract translation keys
npx i18n-extractor

# Upload to Google Sheets (optional)
npx i18n-sheets upload

# Download translations (optional)
npx i18n-sheets download
```

## Migration Guide

### From Previous Versions

This is the first major release. No migration needed.

### Getting Started

1. **Install the package:**

   ```bash
   npm install -g i18nexus-tools
   ```

2. **Initialize your project:**

   ```bash
   npx i18n-sheets init
   ```

3. **Start using the tools:**
   ```bash
   npx i18n-wrapper
   npx i18n-extractor
   ```
